# AGV自动化操作和调度系统

## 功能概述

本系统实现了三维烟厂仓库中小车的自动化操作和任务调度功能。

### 1. 小车自动化操作

#### 1.1 充电站四个充电口
- 每个充电站支持上下左右四个充电口
- 小车空闲状态（`agv_tasks项=0`）自动返回充电口

#### 1.2 电量管理
- **移动速度**: 每秒移动速度固定（默认1m/s）
- **转弯开销**: 转弯需要额外开销（默认0.5秒）
- **电量消耗**: 小车不在充电口时每秒耗电（默认0.5%）
- **预警电量**: 根据返回路程与每秒耗电自动计算

#### 1.3 任务拒绝逻辑
- 有任务分配时，如果计算估计完成后电量低于预警电量，会拒绝执行

## 2. 任务调度器

### 2.1 调度工单类型
- **待入库** (`pending_in`): 分配给空闲小车执行入库任务
- **待出库** (`out_progress`): 分配给空闲小车执行出库任务  
- **待调拨** (`stored`): 分配给空闲小车执行调拨任务

### 2.2 调度流程
1. 定时扫描待处理工单（每秒执行）
2. 筛选空闲小车
3. 计算任务所需电量
4. 检查小车完成任务后是否低于预警电量
5. 如果拒绝，尝试分配给其他小车
6. 分配成功后更新工单状态

## 3. 3D模型

### 3.1 入库口
在3D仓库模型中创建了4个入库口：
- **上方入库口**: 位置 (0, -25)
- **下方入库口**: 位置 (0, 25)
- **左侧入库口**: 位置 (-45, 0)
- **右侧入库口**: 位置 (45, 0)

### 3.2 小车动画
- **运输动画**: 小车从起点移动到终点
- **搬运动画**: 小车到达后执行装卸货（1秒）

## 数据库修改

执行SQL迁移文件添加新表和字段：
```bash
psql -U postgres -d tobacco_warehouse_db -f project-desc/database/sql/agv_automation_tables.sql
```

### 新增表
1. `warehouse_entry_port` - 入库口表
2. `agv_position_history` - AGV位置历史表

### 修改表
1. `charging_station` - 增加四个充电口字段
2. `agv_status` - 增加位置、电量参数等字段
3. `agv_task` - 增加任务估算字段

## 后端API

### AGV调度服务
- `AgvSchedulerService` - 调度核心服务
- `AgvSchedulerTask` - 定时任务（每秒执行）

### 入库口管理
- `WarehouseEntryPortService` - 入库口服务

## 前端功能

### 3D视图
- 入库口可视化
- AGV实时位置显示
- 运输和搬运动画

## 使用说明

1. 执行SQL迁移创建所需表和字段
2. 启动后端服务
3. 启动前端服务
4. 在3D仓库视图中查看入库口
5. 创建工单后，系统自动分配给空闲小车
6. 小车自动执行任务并更新状态
