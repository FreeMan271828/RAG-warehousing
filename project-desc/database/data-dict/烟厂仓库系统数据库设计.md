# 三维烟厂仓库系统数据库设计文档

## 文档信息
- 项目名称：延安烟厂三维数字孪生系统
- 版本：1.0
- 创建日期：2025年1月
- 数据库：MySQL 8.0 / SQL Server

---

## 一、数据库设计概述

### 1.1 设计原则
- 遵循第三范式(3NF)，减少数据冗余
- 保证数据完整性和一致性
- 考虑系统扩展性
- 适配Spring Boot + Vue前后端分离架构

### 1.2 命名规范
- 表名：采用小写字母+下划线格式，如 `sys_user`
- 字段名：采用小写字母+下划线格式，如 `user_name`
- 主键：采用 `id` 命名，自增
- 外键：采用 `表名_id` 格式，如 `role_id`
- 索引：采用 `idx_字段名` 格式
