# 烟厂仓库设备管理知识库

## 设备保养指南

### 如何进行设备日常保养？

1. **清洁设备表面**：使用干净的软布擦拭设备表面，去除灰尘和污垢
2. **检查运行状态**：观察设备指示灯是否正常，听是否有异常声响
3. **记录保养情况**：填写保养记录表，记录保养时间和内容

### 设备定期保养周期

- **日常保养**：每天进行，包括清洁和基本检查
- **周保养**：每周进行一次全面检查
- **月保养**：每月进行一次深度保养
- **年保养**：每年进行一次大修

## 电池管理

### 电池充电注意事项

1. **使用原装充电器**：确保使用与电池匹配的充电器
2. **保持通风**：充电时确保环境通风良好
3. **避免过充**：充满电后及时断开电源
4. **温度控制**：充电时温度不宜过高或过低

### 电池故障处理

1. **充电异常**：检查充电器和电源连接
2. **续航下降**：可能是电池老化，需要更换
3. **发热严重**：立即停止使用并联系维修

## 设备报修流程

### 如何提交报修？

1. 登录系统后进入"设备管理"
2. 选择"报修管理"->"新增报修"
3. 填写设备信息、故障描述
4. 提交后等待维修人员分配

### 报修注意事项

- 描述故障现象要详细准确
- 提供设备位置信息
- 留下联系方式

## 零件库存管理

### 库存预警

当零件库存低于最小库存时，系统会自动预警，提醒管理人员及时采购。

### 零件更换流程

1. 申请零件更换
2. 审批通过后领取零件
3. 进行设备维修更换
4. 记录更换信息
